# Compiler

This is a compiler for a simple language (TBA: definition of said language)

## Building

1. Install Rust from the [rustup page](https://rustup.rs/)
    ```shell script
    curl -sSf https://sh.rustup.rs | sh
    ```

2. Run Makefile
    ```shell script
    make
    ```

3. You can now run the executable with `./compiler`
